# Kebijakan Privasi Aplikasi Mobile TND System

**Terakhir Diperbarui:** 16 Oktober 2025

---

## 1. Pendahuluan

Selamat datang di TND System. Kami berkomitmen untuk melindungi privasi Anda dan memastikan keamanan informasi pribadi Anda. Kebijakan Privasi ini menjelaskan bagaimana kami mengumpulkan, menggunakan, mengungkapkan, dan melindungi informasi Anda saat menggunakan aplikasi mobile TND System.

Dengan menggunakan aplikasi ini, Anda menyetujui pengumpulan dan penggunaan informasi sesuai dengan kebijakan ini.

---

## 2. Informasi yang Kami Kumpulkan

### 2.1 Informasi Pribadi

- **Saat Registrasi:**
  - Nama lengkap
  - Email
  - Nomor telepon
  - ID karyawan
  - Divisi/Departemen
  - Foto profil (opsional)

### 2.2 Data Kunjungan

- Tanggal dan waktu kunjungan
- Lokasi outlet yang dikunjungi
- Waktu check-in dan check-out
- Status kunjungan
- Jawaban checklist
- Catatan dan komentar
- Foto dokumentasi

### 2.3 Informasi Lokasi

Dengan izin Anda, kami mengumpulkan:
- Koordinat GPS saat check-in/check-out
- Digunakan untuk verifikasi kunjungan
- Dikumpulkan hanya saat kunjungan aktif

### 2.4 Informasi Perangkat

- Model dan merek perangkat
- Versi sistem operasi
- Identifikasi perangkat unik
- Versi aplikasi
- Alamat IP

### 2.5 Data Penggunaan

- Fitur yang diakses
- Waktu penggunaan aplikasi
- Frekuensi kunjungan
- Laporan kesalahan

---

## 3. Cara Kami Menggunakan Informasi

### 3.1 Fungsi Utama

- Membuat dan mengelola akun pengguna
- Menjadwalkan dan menugaskan kunjungan
- Merekam data checklist
- Menyimpan foto dokumentasi
- Menghitung skor dan rating

### 3.2 Pelaporan

- Membuat laporan manajemen
- Ekspor data (Excel, PDF)
- Statistik dashboard
- Analisis performa

### 3.3 Komunikasi

- Notifikasi penugasan kunjungan
- Pengingat kunjungan
- Update sistem
- Pengumuman penting

### 3.4 Perbaikan Aplikasi

- Analisis penggunaan aplikasi
- Memperbaiki bug
- Mengembangkan fitur baru
- Meningkatkan pengalaman pengguna

---

## 4. Berbagi Informasi

Kami TIDAK menjual atau menyewakan informasi pribadi Anda. Informasi hanya dibagikan kepada:

### 4.1 Dalam Organisasi Anda

- **Supervisor dan Manajer:** Akses ke laporan kunjungan dan data performa
- **Administrator Sistem:** Mengelola akun dan konfigurasi

### 4.2 Penyedia Layanan

- **Cloud Storage:** Menyimpan foto secara aman (Cloudinary, AWS, dll)
- **Analytics:** Monitoring performa aplikasi

### 4.3 Persyaratan Hukum

Jika diwajibkan oleh hukum atau perintah pengadilan.

---

## 5. Keamanan Data

### 5.1 Enkripsi

- Data saat transmisi: HTTPS/SSL
- Password: Hashing satu arah (bcrypt)
- Data sensitif: Penyimpanan terenkripsi

### 5.2 Kontrol Akses

- Izin berbasis peran
- Autentikasi multi-faktor (jika diaktifkan)
- Audit akses berkala

### 5.3 Pemberitahuan Pelanggaran Data

Jika terjadi pelanggaran data, kami akan:
1. Menyelidiki segera
2. Memberitahu pengguna dalam 72 jam
3. Melaporkan ke otoritas terkait
4. Mengambil tindakan korektif

---

## 6. Hak dan Pilihan Anda

### 6.1 Akses dan Pembaruan

- Lihat informasi profil Anda
- Perbarui detail profil
- Ubah password
- Lihat riwayat kunjungan

### 6.2 Penghapusan Data

- Minta penghapusan akun melalui pengaturan atau administrator
- Data akan dihapus dalam 30 hari
- Beberapa data mungkin disimpan untuk keperluan hukum/bisnis

### 6.3 Kontrol Pengumpulan Data

- **Layanan Lokasi:** Aktifkan/nonaktifkan di pengaturan perangkat
- **Akses Kamera:** Berikan/cabut izin kamera
- **Notifikasi:** Kelola preferensi notifikasi

### 6.4 Portabilitas Data

- Minta salinan data Anda dalam format terstruktur
- Ekspor laporan kunjungan dan foto

---

## 7. Privasi Anak

Aplikasi ini tidak ditujukan untuk anak di bawah 16 tahun. Kami tidak dengan sengaja mengumpulkan informasi dari anak-anak.

---

## 8. Penyimpanan Data

**Periode Penyimpanan:**
- **Akun Pengguna:** Akun aktif + 1 tahun setelah penghapusan
- **Data Kunjungan:** 5 tahun untuk kepatuhan dan audit
- **Foto:** Sama dengan retensi data kunjungan
- **Log:** 90 hari untuk troubleshooting

## 11. Fitur Khusus

### 11.1 Kamera dan Foto

- Aplikasi meminta izin kamera untuk dokumentasi kunjungan
- Foto digunakan hanya untuk verifikasi dan pelaporan
- Foto disimpan dengan aman di server/cloud storage
- Hanya dapat diakses oleh personel berwenang
- Anda dapat meminta penghapusan foto tertentu

### 11.2 Mode Offline

- Data dapat disimpan sementara di perangkat saat offline
- Data disinkronkan ke server saat koneksi pulih
- Data lokal dilindungi oleh fitur keamanan perangkat
